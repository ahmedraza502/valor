## ğŸ“‹ Procurement Workflow

### Step 1: Create Purchase Request
- Choose supplier type (Local or Import)
- Different forms based on supplier type:
  
  **Local PO Fields:**
  - SN#, Product, Supplier, Manufacturer
  - Qty, Rate, Station
  - Payment Terms, Tax, Total
  
  **Import PO Fields:**
  - SN#, Product, Supplier, Manufacturer, HS Code
  - Qty, Rate, Origin
  - Payment Terms, DA/F Payment
  - Dispatched From, Dispatched In, Validity Indent

### Step 2: QC Inspection
- QC team receives products from supplier
- Inspects each item in the purchase order
- Can accept or reject items
- For each item:
  - Accepted quantity and value calculated
  - Rejected quantity and value calculated
  - Rejection reason (if rejected)
  - Remarks
- Separate QC report generated for each PO
- Rejected items tracked for return to supplier

### Step 3: Receipt Generation
- Admin views the QC report
- Can generate two types of receipts:
  1. **Accepted Items Receipt** - For inventory
  2. **Rejected Items Receipt** - For return to supplier
- Each receipt shows:
  - Total quantity
  - Total value
  - Generated by (admin name)
  - Date and remarks
